AuthRecorder Pro Help

Getting Started

AuthRecorder Pro helps you capture and analyze authentication flows. Here's how to use it:

  1. Click the extension icon to open the popup interface
  2. Click "Start Recording" before beginning the authentication process
  3. Complete the login or authentication flow normally
  4. Click "Stop Recording" to save the session

Features

  • Request Capture: Records all HTTP requests during authentication
  • Cookie Monitoring: Tracks cookie changes and session data
  • Token Detection: Automatically identifies authentication tokens
  • Stealth Mode: Prevents detection of automated testing

Stealth Settings

Configure anti-detection features:

  • Browser Profile: Emulate different browser environments
  • Fingerprint Protection: Prevent browser fingerprinting
  • Canvas Noise: Add subtle noise to canvas fingerprints
  • Automation Hiding: Hide automation indicators

Export Options

Export your captured sessions in different formats:

  • JSON - Raw capture data
  • Python Script - Requests-based replay script
  • Playwright Script - Browser automation script
  • Selenium Script - WebDriver-based replay script
